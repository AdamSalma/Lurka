# Lurka

> A unified app for viewing online messaging boards.


## Set-up:
#### Installation
```bash
git clone https://github.com/AdamSalma/Lurka.git

# Install dev dependencies
cd Lurka && npm install

# Install app dependencies
cd app && npm install
```
#### Development mode:
```bash
npm run dev
```
#### Production mode:
```bash
npm run bundle && npm start
```
