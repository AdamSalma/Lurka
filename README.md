# Lurka

> A unified app for viewing online messaging boards.


## Set-up:
#### Installation
```
git clone https://github.com/AdamSalma/Lurka.git

# Install dev dependencies
cd Lurka && npm install

# Install app dependencies
cd app && npm install
```
#### Development mode:
```
# Run from project root (/Lurka/)
npm start
```
#### Production mode:
```
npm run bundle
cd app && npm start
```
